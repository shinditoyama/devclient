# Devclient

Sistema de gerenciamento de clientes (CRUD Básico).

## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

**Para gerar o excutável, sem erro, faça o downgrade do electron-builder para electron-builder@24.6.3**

```bash
# For windows
$ npm run build:win

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```
